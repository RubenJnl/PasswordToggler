PasswordToggler

PasswordToggle is a micro library that toggle's the visability of an password field.
The user can if not sure, easily check if the typed password is correct.
Vitaly Friedman, also mentioned the importance of just one password field on you register form during his talk about Smart eCommerce Checkout Design Patterns at front /> Display/hide the given password.
